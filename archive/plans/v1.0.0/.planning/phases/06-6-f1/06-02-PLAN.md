```python
class TestStep:

```

2. **TestFlow class**:
   - `__init__(self, overlay: TestOverlay, logger: TestLogger)`
   - `load_scenario(self, scenario: TestScenario)`: Load scenario to run
   - `start(self)`: Begin test flow (shows "准备就绪？", waits for F1)
   - `next_step(self)`: Advance to next step (called on F1)
   - `record_feedback(self, feedback: str)`: Record Y/N/SKIP for current step
   - `skip_step(self)`: Skip current step (if allowed)
   - `terminate(self)`: End test early (called on END)
   - `get_current_state() -> dict`: Get current step number, total, instruction

3. **State machine**:
   - States: `IDLE`, `WAITING_FOR_START`, `WAITING_FOR_FEEDBACK`, `COMPLETED`, `TERMINATED`
   - Track current step index
   - Track whether current step's feedback has been recorded

4. **Integration with overlay**:
   - Call `overlay.set_instruction()` to display current instruction
   - Format: "步骤 {n}/{total}: {instruction}\n预期: {expected_result}"
   - Update overlay when state changes

5. **Integration with logger**:
   - Call `logger.start_test()` when test starts
   - Call `logger.log_step()` when step feedback received
   - Call `logger.end_test()` when test completes or terminates

6. **Callback registration**:
   - `setup_hotkeys(self)`: Register F1, END, Y, N callbacks with overlay
   - Handle state transitions in each callback

7. **Error handling**:
   - If no scenario loaded, show error in overlay
   - If step index out of bounds, complete test
   - Log all state transitions

```python
    def test_log_step_records_data(self, logger):
        test_id = logger.start_test("test_scenario")
        step = StepResult(
            step_number=1,
            instruction="Test instruction",
            expected_result="Expected",
            user_feedback="Y",
            timestamp="2026-03-08T14:30:00"
        )
        logger.log_step(test_id, step)
        result = logger.get_test_result(test_id)
        assert len(result.steps) == 1
        assert result.steps[0].user_feedback == "Y"

    def test_end_test_sets_result(self, logger):
        test_id = logger.start_test("test_scenario")
        logger.end_test(test_id, "PASS")
        result = logger.get_test_result(test_id)
        assert result.overall_result == "PASS"
        assert result.end_time is not None
```
