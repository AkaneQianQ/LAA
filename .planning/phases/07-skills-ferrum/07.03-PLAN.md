## Implementation Details

### Ferrum KM API Reference

From documentation (Pages 36-40):

**km.down(key)** - Page 37
- Presses key indefinitely (until km.up() or hardware override)
- key: HID code (0-255)
- Use for hold scenarios or controlled press duration

**km.up(key)** - Page 37
- Releases key force press
- Release lasts 125-175ms (random), then returns to physical state

**km.press(key)** - Page 39
- Full click (press + release) with auto timing
- Press delay: 75-125ms (random)
- Release delay: 125-175ms (random)
- Returns to hardware state after

### HID Key Codes (HUT 1.5)


### Key Name Format

Support these formats:

- Key combinations: "alt+u", "ctrl+shift+a"

**Implement key name parsing**

Add _parse_key() method:
```python
def _parse_key(self, key_name: str) -> List[int]:
    """
    Parse key name or combination into HID codes.

    Args:
        key_name: Key name like 'u', 'alt+u', 'ctrl+shift+a'

    Returns:
        List of HID codes in order
    """
    codes = []
    parts = key_name.lower().split('+')
    for part in parts:
        if part in self.KEY_MAP:
            codes.append(self.KEY_MAP[part])
        else:
            raise ValueError(f"Unknown key: {part}")
    return codes
```

<task id="3">
**Implement press() method**

Add press method for ActionDispatcher:
```python
def press(self, key_name: str) -> None:
    """
    Press a key or key combination.

    Args:
        key_name: Key name (e.g., 'u', 'alt+u', 'enter')
    """
    codes = self._parse_key(key_name)

    if len(codes) == 1:
        # Single key - use km.press for auto timing
        self._send_command(f"km.press({codes[0]})")
    else:
        # Combination - manual down/up sequence
        # Press all modifiers first
        for code in codes:
            self._send_command(f"km.down({code})")
            time.sleep(0.01)  # Small delay between downs

        # Brief hold
        time.sleep(0.05)

        # Release in reverse order
        for code in reversed(codes):
            self._send_command(f"km.up({code})")
            time.sleep(0.01)
```

**Verify:** Single keys and combinations work correctly
</task>

```python
def _order_codes(self, codes: List[int]) -> List[int]:
    """Order codes: modifiers first, then main key."""
    modifiers = {224, 225, 226, 228, 229, 230}  # Ctrl, Shift, Alt
    mod_codes = [c for c in codes if c in modifiers]
    main_codes = [c for c in codes if c not in modifiers]
    return mod_codes + main_codes
```
