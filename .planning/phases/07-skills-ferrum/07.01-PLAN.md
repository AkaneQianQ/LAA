### FerrumController Interface Contract

The controller must implement these methods (as called by ActionDispatcher):
- `click(x: int, y: int) -> None`: Move and click left button
- `press(key_name: str) -> None`: Press a key by name
- `scroll(direction: str, ticks: int) -> None`: Scroll wheel
- `wait(seconds: float) -> None`: Sleep/delay

### Serial Protocol Details

From Ferrum documentation:
- Line terminator: `\r\n` for input, echoed as `\r\n` in output
- Prompt: `>>> ` (three arrows + space, no terminator)
- Command echo: Full command echoed back before result
- Example:
  ```
  Input:  km.init()\r\n
  Output: km.init()\r\n
          >>>
  ```

### Error Handling

- Serial timeout: 1 second default
- Connection failure: raise HardwareConnectionError
- Command timeout: retry once, then raise
