class AccountSwitcher:
    def __init__(self, account_manager: AccountManager):
        self.account_manager = account_manager
        self._workflow_executor: Optional[WorkflowExecutor] = None
        self._is_running: bool = False
        self._lock = threading.Lock()
```

2. **Core methods**:
   - `can_switch() -> bool`
     - Returns True if no workflow is currently running
     - Thread-safe check with lock

   - `switch_to_account(account_hash: str, screenshot: np.ndarray) -> bool`
     - Check can_switch() first
     - Stop current workflow if running (graceful stop)
     - Use AccountManager to switch context
     - Return True on success, False if switch not allowed

   - `switch_to_detected(screenshot: np.ndarray) -> Optional[AccountContext]`
     - Auto-detect account from screenshot
     - Switch to detected account
     - Return new context or None if detection failed

   - `attach_workflow(executor: WorkflowExecutor) -> None`
     - Register workflow executor for lifecycle management
     - Used to check running state and stop safely

   - `on_switch(callback: Callable[[AccountContext, AccountContext], None]) -> None`
     - Callback receives (old_context, new_context)
     - Fired after successful switch

3. **Safety mechanisms**:
   - Lock prevents concurrent switch attempts
   - Workflow must be stopped before switch
   - Immutable context prevents mid-switch corruption
   - Clear error messages for switch failures

4. **Integration with WorkflowExecutor**:
   - Check executor.is_running() or equivalent
   - Call executor.stop() for graceful shutdown
   - Wait for stop_event to be set
   - Create new executor with new account context

5. **Thread safety**:
   - All state changes under lock
   - Callbacks fired outside lock to prevent deadlocks
   - Atomic context swap

def test_no_hardcoded_sleeps_in_core_modules():
    """Verify no time.sleep() calls in core/ directory."""
    excluded_patterns = [
        "# legacy",  # Allow explicitly marked legacy code
        "# SPEED-03-exempt",
        "test_",     # Test files can use sleeps
    ]

    core_dir = Path("core/")
    violations = []

    for py_file in core_dir.rglob("*.py"):
        content = py_file.read_text()
        lines = content.split('\n')

        for i, line in enumerate(lines, 1):
            if 'time.sleep' in line:
                # Check if excluded
                if not any(pat in line for pat in excluded_patterns):
                    violations.append(f"{py_file}:{i}: {line.strip()}")

    assert len(violations) == 0, f"Found time.sleep calls: {violations}"
```
